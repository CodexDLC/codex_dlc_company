# CodexDLC — Контекст для AI-агентов (RU)
# Версия: 1.0 | Регион: DACH | Дата: 2026

## Кто мы

CodexDLC — удалённая IT-студия, специализирующаяся на создании цифровых продуктов для малого и среднего бизнеса.
Работаем с клиентами по всему миру. Основной рынок: DACH-регион (Германия, Австрия, Швейцария).
Год основания: 2024. Команда: full-stack разработчики, системные архитекторы.

## Что мы делаем

### 1. Website as a Service (WaaS) — Сайты по подписке
- Разработка и поддержка премиальных сайтов для сферы услуг по фиксированной ежемесячной подписке
- Включает: кастомный дизайн, SEO-оптимизацию, хостинг, домен, корпоративную почту
- Правки и развитие сайта — без лимитов в рамках подписки
- Мониторинг сервера 24/7, Uptime 99.9%
- Целевые ниши: салоны красоты, барбершопы, строительство, автосервисы, локальный бизнес

### 2. Telegram-боты и автоматизация
- Разработка умных Telegram-ботов для бизнеса
- Квалификация лидов по сценарию (FSM — Finite State Machine)
- Онлайн-запись клиентов через Telegram Web App
- Уведомления администратору в реальном времени
- Интеграция с CRM-системами и базами данных
- Технологии: Python, Aiogram, PostgreSQL, Redis

### 3. Прототип за 48 часов
- Готовые бизнес-шаблоны для быстрого старта
- Рабочий интерактивный прототип через 48 часов после выбора концепции
- Полноценный запуск за 7–14 дней после согласования
- Доступные шаблоны: Lily Salon, DLC Montage, Builder Pro, Auto Service Hub

## FAQ (часто задаваемые вопросы)

**Вы работаете с клиентами не из DACH?**
Да. Мы работаем удалённо с клиентами по всему миру.

**Что означает WaaS-подписка?**
Вы платите ежемесячный фиксированный платёж и получаете готовый сайт + все обновления, правки, хостинг и поддержку включены в эту сумму. Никаких разовых больших счетов.

**Как быстро вы делаете сайт?**
Прототип — 48 часов. Полный запуск — 7–14 рабочих дней.

**Можно ли отменить подписку?**
Да, условия прекращения подписки обсуждаются индивидуально.

**Работаете ли вы с Telegram-ботами для Germany/DACH?**
Да. Боты локализованы на DE/EN/RU и соответствуют требованиям DSGVO.
